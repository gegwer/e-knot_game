# Flappy Parrot Flying 🦜

Классическая игра в стиле Flappy Bird, разработанная на Unity. Игра поддерживает управление мышью, клавиатурой и сенсорным экраном.

## 🎮 Как играть

- **ПК**: Нажмите левую кнопку мыши или клавишу ПРОБЕЛ, чтобы взлететь
- **Мобильные**: Коснитесь экрана, чтобы взлететь
- Избегайте труб и набирайте очки!

## 📦 Сборка проекта

Подробная инструкция по сборке находится в файле [BUILD_INSTRUCTIONS.md](BUILD_INSTRUCTIONS.md)

### Быстрый старт:
1. Откройте проект в Unity (2019.4 LTS или выше)
2. File > Build Settings
3. Выберите платформу (WebGL или Android)
4. Нажмите "Build"
