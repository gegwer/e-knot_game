# 🚨 СРОЧНЫЕ ИСПРАВЛЕНИЯ В UNITY

## Проблема 1: Счет не обновляется

### Вариант А (БЫСТРЫЙ - попробуй сначала):
1. Открой Unity
2. В Hierarchy найди **Pipes** prefab (может быть в Assets → Prefabs)
3. Двойной клик чтобы открыть prefab
4. Внутри найди объект с **Box Collider 2D** (Is Trigger ✓)
5. Выбери этот объект
6. Сверху где **Tag** → выбери **Untagged** (или создай тег "ScoreZone")
7. Убедись что **Is Trigger** включен (галочка ✓)
8. Сохрани (Ctrl+S)

### Вариант Б (если не помогло):
1. Найди любую трубу в Hierarchy во время игры (нажми Play)
2. Посмотри какие объекты внутри труб
3. Найди тот что между трубами (триггер для счета)
4. Запомни его название
5. Останови игру
6. Открой Pipes prefab
7. Найди этот объект
8. **Add Component → Script → ScoreTrigger** (наш скрипт)
9. Сохрани

## Проблема 2: Счетчик посередине экрана мешает

### Решение:
1. В Hierarchy найди **InGameCanvas**
2. Внутри найди **ScoreTextWhite** (текст со счетом)
3. Выбери его
4. Справа в Inspector найди **Rect Transform**
5. Измени позицию:
   - **Anchor Preset**: нажми квадрат вверху справа
   - Выбери **TOP CENTER** (верхний центр)
   - Pos X = 0
   - Pos Y = -100 (это отступ от верха, можешь менять)
6. Увеличь размер текста:
   - **Font Size** = 72 (или больше)
7. Добавь тень для читабельности:
   - **Add Component → Shadow**
   - Effect Distance: X=3, Y=-3

## Проблема 3: Нет кнопки выхода в главное меню

### Решение:
1. Выбери **InGameCanvas** в Hierarchy
2. Правый клик → **UI → Button**
3. Назови кнопку **"HomeButton"**
4. Настрой позицию:
   - **Anchor Preset**: TOP LEFT (верхний левый угол)
   - Pos X = 70
   - Pos Y = -70
   - Width = 80
   - Height = 80
5. Измени текст кнопки на "HOME" или "🏠"
6. В Inspector у кнопки **Button (Script)**:
   - Найди **OnClick()**
   - Нажми **+**
   - Перетащи **Game_Manager** объект в поле
   - Выбери функцию: **Game_Manager → Home()**

## Проблема 4: Счетчик не обновляется (дополнительная проверка)

### В консоли Unity должны быть сообщения "Score added!"
1. Нажми Play
2. Пролети через трубу
3. Внизу окно **Console** (Window → General → Console если не видно)
4. Должно быть: "Score added! Current score: 1"

### Если сообщения нет:
1. Проверь что у **Flappy Bird** (птичка):
   - **Tag = Player** (вверху Inspector)
   - Есть **Rigidbody2D**
   - Есть **Collider2D**
2. Проверь что у триггера в трубах:
   - **Is Trigger** включен ✓
   - **Box Collider 2D** есть
   - Размер коллайдера покрывает промежуток между трубами

---

## БЫСТРЫЙ ТЕСТ:

1. Нажми **Play**
2. Открой **Console** (Ctrl+Shift+C)
3. Пролети через трубу
4. Смотри в консоль - должно быть "Score added!"

Если видишь это сообщение, но счет не показывается на экране:
- Проблема в UI (счетчик не на месте или текст не обновляется)
- Следуй инструкции "Проблема 2" выше

Если НЕ видишь сообщение:
- Проблема в коллайдере/триггере
- Следуй инструкции "Проблема 1" выше
